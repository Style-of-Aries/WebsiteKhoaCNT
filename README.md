Student Management System

Tổng quan

Hệ thống quản lý sinh viên phục vụ các nghiệp vụ cơ bản trong đào tạo như quản lý người dùng, môn học, lớp học phần, điểm danh và kết quả học tập.

Dự án mô phỏng một hệ thống thực tế với phân quyền người dùng và các ràng buộc dữ liệu.

Công nghệ sử dụng

- PHP (lập trình hướng đối tượng - OOP)
- MySQL
- HTML, CSS, JavaScript
- XAMPP

Chức năng chính

Quản lý người dùng

- Tạo, chỉnh sửa, xóa tài khoản
- Phân quyền theo vai trò (5 roles)

Quản lý đào tạo
- Quản lý khoa, môn học, lớp hành chính
- Quản lý lớp học phần theo học kỳ
- Kiểm tra ràng buộc dữ liệu khi thao tác

Điểm danh và kết quả học tập
- Điểm danh theo từng buổi học (có mặt, vắng, muộn)
- Nhập và quản lý điểm
- Tra cứu kết quả học tập

Chức năng sinh viên
- Xem hồ sơ cá nhân
- Xem thời khóa biểu
- Xem kết quả học tập
- Đăng ký lớp học phần

Thiết kế hệ thống
- Cơ sở dữ liệu quan hệ (MySQL)
- Áp dụng OOP trong PHP
- Phân quyền theo Role-Based Access Control (RBAC)
- Kiểm soát logic và dữ liệu ở backend

Cài đặt

- Clone project
git clone https://github.com/Style-of-Aries/WebsiteKhoaCNT.git
- Database
Import file system_services.sql trong folder \database vào MySQL (phpMyAdmin)
- Chạy project
Copy vào thư mục www (Laragon)
Truy cập:
http://localhost/WebsiteKhoaCNT
